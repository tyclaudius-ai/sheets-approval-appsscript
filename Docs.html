Sheets Approvals — Quick Help

This sidebar is part of the copy/paste MVP (no Marketplace add-on required).
Fast demo:
  1. Run Approvals → Create demo setup
  2. Go to Requests, click a row
  3. Run Approvals → Approve row
  4. Check Audit for the appended event + hash

Re-approval on change

  • If an APPROVED row is edited, it can auto-revert to PENDING and log REAPPROVAL_REQUIRED.
  • Configure tracked/exempt columns in CFG.REAPPROVAL_TRACKED_HEADERS and CFG.REAPPROVAL_EXEMPT_HEADERS.

Row locking (optional)

  • On approval, the script can apply a row protection (warning-only by default).
  • Toggle via CFG.LOCK_ROW_ON_APPROVE and CFG.LOCK_WARNING_ONLY.
