Microproduct • Apps Script

Google Sheets Approvals + Audit Trail

A copy/paste Apps Script template that adds Approve/Reject actions and an append-only audit log to any Google Sheet.

Read the README One‑pager Setup checklist Demo template Screenshot checklist

No add-ons. No external services. Runs in your Google Workspace.

What you get

  • Custom menu: Approvals → Approve / Reject
  • Append‑only Audit sheet (timestamp, user, action, row id, snapshot hash)
  • Optional “row lock” warning to prevent approving stale data
  • Docs + outreach copy for selling/implementing

Fits use cases like

  • Expense approvals
  • Content approvals
  • Engineering change requests
  • Vendor onboarding checklists

How it works (high level)

  1. Paste Code.gs into your Sheet’s Apps Script project
  2. Reload the Sheet
  3. Select a row → Approve/Reject
  4. Audit log records who did what + when

Demo checklist

  • Paste Code.gs into your Sheet’s Apps Script project
  • Reload the Sheet; run Approvals → Create demo setup
  • Approve a seeded request; verify an Audit entry appears
  • Edit an approved row; verify it reverts to PENDING + logs REAPPROVAL_REQUIRED
